# caderno-tem-tico-no-NotebookLM
Projeto com o objetivo de criar um caderno temático no NotebookLM. Com o tema de Fundamentos e dinâmicas do Mercado Financeiro Brasileiro, voltado para iniciantes, com fontes abertas em texto. Foi utilizado o bloco de notas para elaborar o objetivo inicial e a coleta de dados de vídeo e texto no formato a baixo:



Objetivo:

Criar um segundo cérebro que explique de forma simples a lógica de funcionamento do mercado financeiro para iniciantes.
